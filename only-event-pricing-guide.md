# 同人展会定价参考手册

本文档是[《同人市场分析（补）——同人展举办方的无奈》](https://seikasahara.com/zh/posts/gytf/)的实操附录。面向展会主办方，将理论模型的数值模拟结果和参数测量方法整理为可直接使用的参考资料，并附问卷模板。

---

## 一、数值模拟：中小型 only 展的定价参考

### 1.1 场景设定

以一个典型的中小型 IP only 展为例：

| 参数 | 预估值 | 说明 |
|---|---|---|
| 场地面积 | 800 m² | 典型中小场馆（如酒店宴会厅、社区活动中心）|
| 摊位数 | 60 个 | 21 个偏商业型 / 39 个偏交流型 |
| 潜在消费者 | 1000 人 | 目标 IP 圈子在该城市的核心受众 |
| 社团参展成本（商业型）| 800 元 | 高铁往返 + 住宿 + 印刷备货 |
| 社团参展成本（交流型）| 600 元 | 高铁往返 + 住宿 + 少量物料 |
| 场地运营成本 | 10,000 元 | 场地租赁 + 基本运营 |

参展成本反映了中国的地理跨度——多数社团需要跨城参展：高铁往返 400–800 元、住宿 200–400 元、餐饮与物料 200–300 元，商业型社团还需加上印刷备货。

### 1.2 门票定价

模型给出两个极端和一个现实落点：

| 定价策略 | 门票 | 预期入场人数 | 拥挤度 | 每摊位客流 |
|---|---|---|---|---|
| **社会最优**（让所有人最开心）| **19 元** | 700 人 | 1.25 人/m²（舒适）| 11.7 人 |
| **盈亏平衡**（不亏不赚）| **40 元** | 580 人 | 1.04 人/m² | 9.7 人 |
| 含安全边际 | **45 元** | 551 人 | 0.98 人/m² | 9.2 人 |
| 含较大安全边际 | **50 元** | 523 人 | 0.93 人/m² | 8.7 人 |
| **利润最大化**（赚最多钱）| **71 元** | 403 人 | 0.72 人/m²（偏空）| 6.7 人 |

各档位的预期利润：

| 门票 | 门票收入 | 摊位收入 | 净利润 |
|---|---|---|---|
| 19 元 | 13,125 | −16,722 | **−13,597**（亏损）|
| 40 元 | 23,200 | −12,830 | **+370**（刚好打平）|
| 45 元 | 24,795 | −11,853 | **+2,942** |
| 50 元 | 26,150 | −10,948 | **+5,202** |
| 71 元 | 28,613 | −5,571 | **+13,042** |

> **关键发现：盈亏平衡点在 40 元左右。** 现实中大多数 only 展收 40–50 元，恰好对应"不亏钱但也不是为了赚钱"的定价策略。这不是巧合——40 元以下需要主办方自掏腰包补贴，50 元以上消费者开始觉得"太贵了"。40–50 元是唯一同时满足主办方生存和消费者接受度的稳定区间。

### 1.3 摊位费定价

模型建议的摊位费结构是**按社团影响力差异化定价**：影响力大的社团应该获得优惠甚至补贴，影响力小的社团付标准价。

理想情况下的摊位费参考（按社团影响力分档）：

| 社团类型 | 典型画像 | 建议摊位费 | 说明 |
|---|---|---|---|
| 普通商业型（影响力低）| 新刊贩卖为主的小社团、代工周边摊、初次参展的个人画师 | 约 350 元 | 标准价 |
| 普通交流型（影响力低）| 无料配布社团、交换本社团、手工赠品摊、纯展示型摊位（画板/手作展示）、Coser 交流摊 | 约 110 元 | 氛围贡献换来的折扣 |
| 中等商业型（有一定粉丝）| 在 Pixiv/Lofter 有稳定读者的连载作者、多次参展有口碑积累的社团、小有名气的周边制作者 | 免费或小额补贴 | 引流价值开始抵消坑位成本 |
| 中等交流型 | 圈内知名的无料大手、长期活跃的安利/考据/翻译社团、组织过线上企划的社群核心成员 | 补贴约 170 元 | 引流 + 氛围双重贡献 |
| 知名社团（大手）| 粉丝数万级的头部创作者、排队必排的商业级同人社团、自带话题性的传奇社团 | 补贴 500–2000+ 元 | 引流价值远超坑位成本 |

> **为什么要补贴大手？** 一个知名社团的到场可以额外吸引几十甚至上百名消费者，每人贡献 40–50 元门票收入。补贴 1000 元换来 20 个额外消费者（每人 45 元门票 = 900 元），几乎能打平——更不用说氛围提升带来的间接效益。

> **为什么交流型更便宜？** 交流型社团提供无料配布、互动展示、同好交流，这些构成了展会氛围的核心。他们的"贡献"不体现在摊位费上，而体现在整个展会的体验质量上。给他们降价不是照顾，是为展会买氛围。

> **关于"商业型"和"交流型"的区分**：这不是非黑即白的二分法，而是一个连续谱。同一个社团可能既卖本也发无料，判断标准是**主要动机**——如果抽掉销售收入他们就不会来，偏商业型；如果不卖东西也想来交朋友，偏交流型。社团问卷中的"如果一件都没卖出去但交到了朋友，值得吗？"这道题就是用来区分的。

### 1.4 录取门槛

跨城参展成本意味着并非所有社团都适合来参展：

| 社团类型 | 最低影响力门槛 | 含义 |
|---|---|---|
| 商业型 | 中等偏上 | 影响力不到中等的商业型小社团跑外地展是亏钱的 |
| 交流型 | 几乎无门槛 | 成本更低 + 社交回报 + 氛围补贴，绝大多数都值得来 |

这与"小社团跑外地展不划算"的普遍体感一致。

### 1.5 利润最大化的代价

如果主办方纯粹追求利润最大化（门票 71 元），会发生什么：

- 入场人数从 700 降到 403（减少 42%）
- 每摊位客流从 11.7 降到 6.7（社团收入缩水）
- 拥挤度从 1.25 降到 0.72（场馆偏空）
- 社团客流回报（570 元）不够覆盖参展成本——主办方反而需要**倒贴全部社团**
- 利润完全来自门票

这种"门票贵、人少、空荡荡"的体验，恰好是消费者吐槽商业展的典型特征。

## 二、如何测量模型参数

模型涉及的参数中，有些可以直接算出来，有些需要通过问卷和数据间接估计。以下按难度分类。

### 2.1 可直接测量的参数

| 参数 | 怎么测 |
|---|---|
| 场地面积、摊位数 | 直接测量 |
| 运营成本 | 直接核算 |
| 入场人数 | 售票系统 / 入口计数 |
| 社团参展成本 | 社团问卷（见下文）|

### 2.2 有自然代理变量的参数

**社团影响力（模型中的 θ）**

这是最容易量化的"软参数"。推荐使用社交媒体指标作为代理：

- Pixiv 作品总收藏数
- 微博 / Twitter 粉丝数
- Lofter 热度
- 过往出摊时的排队长度（如有记录）

操作方法：将所有申请社团的指标收集后，取中位数作为基准（θ = 1），其他按比例缩放。如果数据来自多个平台，可以取各平台排名的几何平均。

**社团类型（商业型 vs 交流型）**

社团类型是一个连续谱，不是二分变量。以下代理指标可以组合使用，综合判断一个社团在谱上的位置：

**摊位申请阶段可观测的指标：**

- **无料率**——申请表中申报的出品清单里，无料（免费配布）与贩卖品的数量比。无料率高 → 偏交流型。这是最直接的单一指标。
- **申请动机文本**——申请表中"参展理由"或"社团介绍"的措辞。提到"宣传新刊""回本""销售目标"→ 偏商业型；提到"交流""交朋友""分享""为爱发电"→ 偏交流型。关键词频率可以量化。
- **定价策略**——出品定价是否覆盖成本并留有利润空间。按成本价或低于成本价出售 → 偏交流型；明确的利润定价 → 偏商业型。
- **产品形态**——手工少量制作（手绘明信片、手作周边）→ 偏交流型；标准化批量生产（印刷本、工厂代工周边）→ 偏商业型。
- **参展频率与选择性**——广泛报名各地展会、只去客流大的展 → 偏商业型；只参加特定 IP/圈子的 only 展、有社群归属感 → 偏交流型。

**社交媒体可观测的指标：**

- **内容类型**——日常发布以预售/宣传/通贩链接为主 → 偏商业型；以创作过程、同好互动、二创讨论为主 → 偏交流型。
- **互动模式**——主要与粉丝单向互动（回复购买咨询）→ 偏商业型；与同圈创作者频繁双向互动（互转、联动、合志邀约）→ 偏交流型。

**历史数据指标（需要积累）：**

- **销售波动下的参展留存**——过去某次展会销售很差后，下次还来吗？来 → 偏交流型（参展动机不完全依赖收入）；不来 → 偏商业型。
- **展会问卷中的关键题**——"如果一件都没卖出去但交到了朋友，值得吗？"回答"完全值得"或"基本值得"→ 交流型；"不太值得"或"完全不值得"→ 商业型。

**实操建议**：如果只能用一个指标，用**无料率**。如果能用两个，加上**问卷动机题**。三个以上指标时，可以做简单加权评分（例如：无料率 ×40% + 动机题得分 ×30% + 定价策略 ×30%），得到 0–1 的连续类型分数，0 = 纯商业，1 = 纯交流。

**社团客流回报**

需要跨展会积累数据：记录每次展会的平均每摊位销售额和入场人数，回归拟合即可。非货币回报（社交、认同感）可通过社团问卷标定比例。

### 2.3 需要问卷估计的参数

**消费者异质性（保留价格分布）**

最佳测量方法是**阶梯定价实验**：同一展会设多档价格（早鸟 / 预售 / 现场），各档购买人数的比例直接反映消费者的支付意愿分布。

**拥挤敏感度**

需要跨期比较：同一展会系列在不同场地举办时，消费者满意度如何随拥挤度变化。问卷中需要同时收集"对拥挤的感受"和整体满意度。

**氛围价值（最难测量）**

氛围是集体现象，没有直接测量手段。推荐组合使用：

1. **事后问卷**——直接问"氛围怎么样"（见下文模板）
2. **行为指标**——停留时长（氛围好 → 待得久）、社交媒体发布率（氛围好 → 更多现场分享）
3. **对照比较**——在不同展次比较社团构成与满意度的关系

**社交回报敏感度**

核心问题："如果一件都没卖出去，但交到了新朋友，这次参展值得吗？"（见社团问卷）

## 三、问卷模板

以下是三套问卷模板，分别用于消费者（事后）、社团（事后）、消费者（事前/预售阶段）。主办方可以根据需要裁剪使用。

### 3.1 消费者事后问卷

> 展会结束后 1–3 天内发放，目标是收集本次体验数据。

**基本信息**

1. 您是第几次参加本展会？
   - [ ] 第一次
   - [ ] 第二次
   - [ ] 第三次及以上

2. 您从哪里来参加本展会？
   - [ ] 本市
   - [ ] 省内其他城市
   - [ ] 外省

3. 您本次参展的总花费大约是多少（含门票、交通、餐饮、购买同人作品）？
   - ______ 元

**门票与支付意愿** → 用于估计消费者异质性

4. 本次门票价格是 _____ 元。您觉得这个价格：
   - [ ] 很便宜，加价 50% 也会来
   - [ ] 合理，大致愿意接受
   - [ ] 偏贵，但还是来了
   - [ ] 很贵，犹豫了很久才决定来

5. 如果门票涨到以下价格，您还会来吗？（每行选是/否）

   | 价格 | 会来 | 不会来 |
   |---|---|---|
   | 当前价格 +10 元 | [ ] | [ ] |
   | 当前价格 +20 元 | [ ] | [ ] |
   | 当前价格 +30 元 | [ ] | [ ] |
   | 当前价格 +50 元 | [ ] | [ ] |

**拥挤感受** → 用于估计拥挤敏感度

6. 您对本次展会的拥挤程度感受如何？
   - [ ] 太空了，冷冷清清
   - [ ] 刚好，可以自由走动
   - [ ] 有点挤但可以接受
   - [ ] 很挤，影响了体验
   - [ ] 非常挤，严重影响了体验

7. 如果同样的展会内容但人多一倍（更挤），您的参加意愿会：
   - [ ] 不受影响
   - [ ] 稍微降低
   - [ ] 明显降低
   - [ ] 可能不来了

**氛围评价** → 用于估计氛围参数

8. 请为以下方面打分（1–10 分）：

   | 方面 | 分数 |
   |---|---|
   | 整体氛围 | _____ |
   | 社团摊位的互动和交流体验 | _____ |
   | 现场观众的热情和友好程度 | _____ |
   | "同好聚会"的感觉 | _____ |

9. 本次展会中，以下哪些体验对您来说最重要？（最多选 3 项）
   - [ ] 购买喜欢的同人作品
   - [ ] 与喜欢的社团面对面交流
   - [ ] 看到 Coser / 拍照
   - [ ] 和朋友一起逛展
   - [ ] 发现新社团 / 新作品
   - [ ] 现场的热闹氛围本身

**吸引力来源** → 用于估计社团影响力

10. 您来参展的最主要原因是什么？（单选）
    - [ ] 冲着特定社团来的
    - [ ] 整体阵容看起来不错
    - [ ] 朋友叫我来的
    - [ ] 固定参加这个系列的展会
    - [ ] 随便逛逛

11. 如果有的话，请列出您**专门为了见到 ta 们而来**的社团名称（最多 3 个）：
    - ______________________

**人流量期望** → 用于估计最低氛围阈值

12. 假设本场地最大容纳量为 _____ 人，您觉得来多少人比较合适？
    - ______ 人左右

### 3.2 社团事后问卷

> 展会结束后发放给参展社团，目标是收集供给侧数据。

**基本信息**

1. 您的社团名称：______________________

2. 您这次带了哪些类型的出品？（可多选）
   - [ ] 贩卖本 / 周边（定价出售）
   - [ ] 无料配布（免费赠送）
   - [ ] 展示 / 交流为主（不以出售为主要目的）

3. 贩卖品与无料的**数量比**大约是？
   - [ ] 全部是贩卖品
   - [ ] 贩卖为主，少量无料
   - [ ] 大约各一半
   - [ ] 无料为主，少量贩卖
   - [ ] 全部是无料 / 纯展示交流

**参展动机** → 用于判断社团类型

4. 您参加本次展会最重要的目的是什么？（单选）
   - [ ] 销售作品获取收入
   - [ ] 宣传自己的作品 / 扩大影响力
   - [ ] 与同好交流、交朋友
   - [ ] 交换无料 / 参与社群活动
   - [ ] 支持这个 IP / 展会

5. 如果这次展会一件作品都没卖出去（或者没有任何交换），但交到了新朋友、聊得很开心，您觉得这次参展：
   - [ ] 完全值得
   - [ ] 基本值得
   - [ ] 有点遗憾但还行
   - [ ] 不太值得
   - [ ] 完全不值得

**参展成本** → 直接测量

6. 本次参展的各项实际花费：

   | 项目 | 金额（元）|
   |---|---|
   | 交通（往返）| ______ |
   | 住宿 | ______ |
   | 餐饮 | ______ |
   | 印刷 / 制作成本 | ______ |
   | 物料运输 | ______ |
   | 摊位费 | ______ |
   | 其他 | ______ |
   | **合计** | ______ |

**客流回报** → 用于估计回报函数

7. 本次展会中，大约有多少人来过您的摊位（停留超过 10 秒）？
   - ______ 人

8. 本次展会的销售额大约是：
   - ______ 元

9. 除了销售额，您从本次参展中获得的**非货币收益**（如认同感、新粉丝、社交）大约值多少钱？（主观估计）
   - [ ] 几乎没有
   - [ ] 大约 100–300 元
   - [ ] 大约 300–500 元
   - [ ] 大约 500–1000 元
   - [ ] 超过 1000 元

10. 综合考虑所有成本和收益（包括非货币的），本次参展总体来说：
    - [ ] 赚了不少
    - [ ] 小赚或基本打平
    - [ ] 小亏但可以接受
    - [ ] 亏了不少
    - [ ] 严重亏损

**对展会的评价** → 用于估计氛围和拥挤

11. 请为以下方面打分（1–10 分）：

    | 方面 | 分数 |
    |---|---|
    | 到场消费者的数量 | _____ |
    | 消费者与摊位的互动质量 | _____ |
    | 展会整体氛围 | _____ |
    | 场地拥挤程度的合理性 | _____ |

12. 如果下次同一展会的入场人数增加 50%，对您来说：
    - [ ] 好事，更多潜在客户/交流对象
    - [ ] 无所谓
    - [ ] 不太好，太挤了没法好好交流

### 3.3 消费者事前问卷（预售阶段）

> 在展会宣传期 / 预售期发放，用于提前了解消费者构成和需求。

1. 您对本次展会的参加意愿如何？
   - [ ] 一定会去
   - [ ] 大概率会去
   - [ ] 看情况
   - [ ] 大概率不去
   - [ ] 完全不会去

2. 以下哪个因素对您的参加决策影响最大？（单选）
   - [ ] 有特别想见的社团
   - [ ] 门票价格
   - [ ] 交通是否方便
   - [ ] 朋友是否同行
   - [ ] 展会的口碑 / 往期评价

3. 您愿意为本次展会支付的最高门票价格是：
   - ______ 元

4. 以下社团中，哪些是您**专门想去见的**？（展示已确认参展的社团列表，可多选）
   - [ ] 社团A
   - [ ] 社团B
   - [ ] ...

## 四、数据使用建议

### 4.1 最低数据量

- **消费者问卷**：至少回收 50 份有效问卷才能得到有意义的统计结论。回收率通常 10–20%，建议发放 300+ 份。
- **社团问卷**：全部参展社团都应填写。60 个摊位 = 60 份数据。
- **跨期积累**：单次展会只能得到一个数据点。多数参数需要至少 3–5 次展会的数据才能稳定估计。

### 4.2 定性结论不需要精确参数

模型最重要的结论对参数精度**不敏感**：

- **谁应该被优先补贴**：影响力大的交流型社团 > 影响力大的商业型 > 影响力小的交流型 > 影响力小的商业型。这个排序在任何合理参数下都不变。
- **40–50 元是稳定的门票区间**：只要展会有正的外部性（社团的到场让消费者更开心、消费者的到场让社团更满意），盈亏平衡点就必然存在于"完全免费"和"利润最大化"之间。
- **利润最大化总是导致消费者更少**：这是垄断定价的数学必然，与参数无关。

因此，**即使没有精确的问卷数据，主办方也可以直接使用本文的定性建议**。问卷数据的作用是让定价更精准、让补贴更精确、让跨期决策有据可依。

### 4.3 问卷实施流程建议

1. **展前**：发放事前问卷，收集支付意愿和社团吸引力数据。用于设定门票价格和社团邀请优先级。
2. **展中**：记录入场人数（分时段）、场地实际使用面积。
3. **展后 1–3 天内**：发放消费者和社团事后问卷。趁记忆新鲜，回收率最高。
4. **展后 1 周内**：汇总数据，计算本期各参数的估计值，为下期决策提供输入。

长期积累 3–5 期数据后，可以建立该展会系列的"参数档案"，使定价从拍脑袋变为有数据支撑的系统性决策。
